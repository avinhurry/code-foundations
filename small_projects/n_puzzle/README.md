# N-Puzzle solver

Ruby implementation of the N-Puzzle (e.g. 8-puzzle / 15-puzzle) using depth first search with backtracking.

## Problem

An N-Puzzle consists of a square grid (e.g 3x3, 4x4) containing numbers 1 to N and one empty space (represented as 0). The goal is to
transform a scrambled board into the target ordered configuration by sliding tiles into the empty space.
